# Node-GPT
**Command Line Interface for [ChatGPT](https://chat.openai.com) using [Node.JS](https://nodejs.org).**

### Get Started

**Step 1:** Clone the repo and install dependencies.
```
npm install
```

**Step 2:** Run the Node.js server.
```
npm run start
```
